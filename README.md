# CodeCanvas
A simple HTML live server code, where you can write the HTML code in your browser and preview it live on the right side of the browser.

You can type the code on the left and instantly see it on the right side, making it easier to learn and practice HTML Code 


You can Try it out at https://code-canvas-gilt.vercel.app/
